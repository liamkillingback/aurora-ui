# Support

Aurora UI is a community open-source project maintained by PHXTemplates. Here is
where to go for help.

## Questions & usage help

- **GitHub Discussions** → the "Q&A" category. Search first; someone may have
  already answered.
- Check the [documentation](https://aurora-ui.phxtemplates.com), especially
  [Getting started](docs/getting-started.md) and
  [Troubleshooting](docs/troubleshooting.md).

## Bugs

Open a GitHub issue using the **Bug report** template. Include your Elixir,
Phoenix, LiveView, and Tailwind versions, a minimal reproduction, and what you
expected. Accessibility bugs are prioritized.

## Feature & component requests

Open a **Component request** or **Feature request** issue. We prioritize using
observed demand, accessibility risk, maintenance cost, and relevance. See the
public [contribution priorities](docs/roadmap.md).

## Security

See [`SECURITY.md`](SECURITY.md). Never report vulnerabilities in public issues.

## Response targets

These are goals for a small maintainer team, not contractual guarantees:

| Item | Target first response |
|------|----------------------|
| Security report | 3 business days |
| Accessibility regression | 5 business days |
| Bug report | 7 business days |
| Feature/component request | 14 business days |

## Commercial support

Aurora UI itself is free forever. If you want production Phoenix starters built
on the same design language, see [PHXTemplates](https://phxtemplates.com). Buying
a template is never required to use, theme, or extend Aurora UI.
